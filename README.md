# Flabee

Another Flappy Bird clone which is player character is a bee that I made using SDL3 and C11.

## Build
```sh
cmake -S . -B Build -DSDL_STATIC=1
cmake --build Build
```

## Run
```sh
./Build/Flabee
```

## Clean
```sh
rm -rf Build
```
